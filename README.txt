Conversão de Números por Extenso

Este projeto é um estudo que utiliza as biblioteca `numerizer` para converter números por extenso em português para sua forma numérica.

Objetivo:
O principal objetivo deste código é realizar tarefas de conversão de números escritos por extenso. Isso pode ser útil em projetos 
de análise de dados que envolvem a manipulação de informações numéricas.

Como Funciona:
1. Tradução para Inglês: Utiliza a biblioteca `deep_translator`.
2. Conversão Numérica: Utiliza a biblioteca `numerizer`.
3. Exibição de Resultados: Imprime o valor numérico e seu tipo de dado.

Pré-requisitos:
- Instale as bibliotecas necessárias executando: `pip install numerizer deep_translator`.

Como Usar:
1. Clone o repositório.
2. Execute o script Python: `python main.py`.
